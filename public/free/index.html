Deep research on your own API keys, straight from this browser: every model call goes directly to OpenAI or Groq — this site's server is never in the path, so it could not log your messages even if it wanted to. One secret, kept in your password manager, encrypts everything (your chats and your API keys) before anything is stored.

This is your new secret — it is the ONLY way back into this project. Press Open and your password manager will offer to save it; you can also copy it now:

Only providers that allow direct browser calls (CORS) can work this way — today that is OpenAI and Groq. Your keys and chats are sealed in this browser with a key derived from your secret; the server stores one unreadable blob so the project follows you across devices. Losing the secret loses the project — there is no recovery. About Deepresearch
