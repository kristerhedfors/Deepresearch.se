DeepResearch.se

A deep-research AI assistant that plans its own web searches, checks its coverage, and answers with numbered citations — within a time budget you set.

Built over a weekend — entirely from a phone

The assistant is real, but it's also the evidence: this whole SaaS-style application — Google sign-in, research quotas grounded in real cost, a five-phase research pipeline, an admin console, a PWA — was built over one weekend entirely through the Claude Code iPhone app. Every prompt, the domain purchase, every deployment, every service configuration; the source code was never viewed on any other device. The complete prompt-by-prompt record is public, and so is the code.
