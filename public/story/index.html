The build story

← Back to the app

How this site started, built over a weekend entirely through the Claude Code iPhone app. The domain purchase, every deployment, and every service were configured without the source or config ever viewed directly on any other device (one exception: the D1 database UUID had to be copied by hand from the Cloudflare dashboard URL). It records every prompt verbatim, what broke, what was discovered, and the commit that landed each step. That first weekend of ~70 prompts was only the beginning; the project has kept growing in new directions since, and stays experimental. The source is public at github.com/kristerhedfors/Deepresearch.se; what the site is for and its use restrictions are on About this project.
